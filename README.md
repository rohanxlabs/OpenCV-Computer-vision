## 📌 Overview

This repository represents a structured, phase-by-phase journey into Computer Vision using OpenCV.  
It demonstrates strong fundamentals, hands-on implementation, and progressive problem-solving skills.

> ⚡ From basic image operations → to real-time video processing → to face & object detection

---

## 🧠 Project Architecture

OpenCV/ ├── Phase 1 - Basics
├── Phase 2 - Image Resizing & Transformations
├── Phase 3 - Drawing Techniques
├── Phase 4 - Video Processing
├── Phase 5 - Image Filtering & Blurring
├── Phase 6 - Edge Detection & Thresholding
├── Phase 7 - Contours & Shape Detection
├── Phase 8 - Face & Object Detection


---

## 📚 Phase Breakdown

### 🔹 Phase 1: Image Fundamentals
- Image loading and visualization  
- Color space conversion (BGR → RGB → Grayscale)  

### 🔹 Phase 2: Transformations
- Resizing and scaling  
- Cropping techniques  
- Geometric transformations  

### 🔹 Phase 3: Drawing & Annotation
- Drawing primitives (lines, rectangles, circles)  
- Adding text overlays  

### 🔹 Phase 4: Video Processing
- Real-time webcam capture  
- Frame-by-frame processing pipeline  

### 🔹 Phase 5: Filtering & Blurring
- Gaussian blur  
- Averaging and smoothing filters  

### 🔹 Phase 6: Edge Detection
- Canny Edge Detection  
- Thresholding techniques  

### 🔹 Phase 7: Contours & Shape Analysis
- Contour detection  
- Shape classification  
- Boundary extraction  

### 🔹 Phase 8: Face & Object Detection
- Haar Cascade-based face detection  
- Basic object detection pipeline
