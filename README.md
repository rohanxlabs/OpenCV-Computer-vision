📸 OpenCV Computer Vision Project (Phase-wise Learning)


🚀 Overview


This repository contains a phase-wise structured journey through Computer Vision using OpenCV (Python).
Each phase focuses on a specific concept, starting from basic image handling to advanced tasks like face and object detection.
This project is designed to demonstrate practical understanding + progressive learning in computer vision.


🧠 Project Structure


OpenCV/
 ├── Phase 1 - Basics
 ├── Phase 2 - Image Resizing & Transformations
 ├── Phase 3 - Drawing Techniques
 ├── Phase 4 - Video Processing
 ├── Phase 5 - Image Filtering & Blurring
 ├── Phase 6 - Edge Detection & Thresholding
 ├── Phase 7 - Contours & Shape Detection
 ├── Phase 8 - Face & Object Detection


📚 Phase Details

Image loading & display
Color formats (BGR, RGB, Grayscale)

🔹 Phase 3: Drawing Techniques

Drawing shapes (line, rectangle, circle)
Adding text on images

🔹 Phase 4: Video Processing

Capturing video using webcam

Real-time display

Gaussian Blur

Averaging
Smoothing techniques

🔹 Phase 6: Edge Detection & Thresholding
Canny Edge Detection

Thresholding techniques

Contour detection

Object boundary extraction

Basic object detection
